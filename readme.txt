Для сборки проекта требуется выполнить скрипт build.bat
Перед запуском нужно проинициализировать базу данных начальными значениями, в папке DB лежит бекап базы.
Папку data скопировать в папку target
Запускается проект скриптом start.bat (по умолчанию search, для выполнения stat - заменить "search" на "stat")
